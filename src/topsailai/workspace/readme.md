# Unified Worker Entry

This workspace package is An Unified Worker Entry.

- xxx_shell.py : A Module that Reference entry used on the command line

## Agent-WorkSpace & Agent-WorkFolder

1. TOPSAILAI_HOME, The Agent-Workspace, refer to `workspace/folder_constants.py`, Files generated by the Agent process, including of `log`, `lock`, `task`, `story_memory` etc..
2. TOPSAILAI_WORK_FOLDER, The working directory of the Agent process, including of `.env`, `db` etc..

## Core Modules

### hook_instruction.py

Hook-based instruction system for command processing.

- **Trigger**: Messages starting with `/` are intercepted as hook commands
- **Hook Management**: Register, delete, check, and execute hooks via `HookInstruction`
- **Plugin Integration**: Auto-loads hooks from `INSTRUCTIONS` registry on initialization
- **Help System**: Built-in `/help` hook to list all registered commands

### input_tool.py

Interactive input utilities with hook integration and history persistence.

- **Input Modes**: Single-line (`input_one_line`) and multi-line (`input_multi_line`) input
- **Hook Interception**: Automatically routes `/` commands through `hook_instruction.py`
- **History Persistence**: Saves and restores readline history to `TOPSAILAI_HOME/.input_history`
- **Exit Handling**: Supports `exit`, `quit`, `/exit`, `/quit` for graceful termination
