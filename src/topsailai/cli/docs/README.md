# Documentation

The `docs/` directory organizes documentation files by single-level subfolders.

## Structure

- Subfolders are one level deep under `docs/` (for example, `usage/` and `memo/`).
- Each subfolder contains `.md` files.
- `.md` files may be regular files or symbolic links.

## Access

These documentation files can be listed and read directly through `topsailai.py` (invoked as the `topsailai` command).
